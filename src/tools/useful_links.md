# Useful Links

This page collects external tools and references commonly useful for X-Ray and S.T.A.L.K.E.R. modding.

## X-Ray Engine and Modding

- [OpenXRay](https://github.com/OpenXRay/xray-16): open-source X-Ray engine fork used by XRF.
- [Anomaly modding book](https://igigog.github.io/anomaly-modding-book/): community documentation for Anomaly-style
  modding concepts.
- [STALKER Anomaly modded exes](https://github.com/themrdemonized/STALKER-Anomaly-modded-exes): useful reference for
  engine-side ideas and DLTX-related behavior.

## Asset Tools

- [AXRToolset](https://github.com/revolucas/AXRToolset): utilities for unpacking and working with gamedata.
- [NVIDIA Texture Tools Exporter](https://developer.nvidia.com/nvidia-texture-tools-exporter): DDS texture import/export
  tooling.
- [xray-skls-file-browser](https://github.com/vika-sonne/xray-skls-file-browser): animation-related file browser and
  converter.

## Development Tools

- [Windows Terminal](https://apps.microsoft.com/store/detail/windows-terminal/9N0DX20HK701): terminal for running build,
  CLI, and validation commands on Windows.
- [TypeScriptToLua](https://typescripttolua.github.io/): TypeScript-to-Lua compiler used by the XRF script build.
- [Tauri](https://tauri.app/): framework used by the XRF tools desktop app.
