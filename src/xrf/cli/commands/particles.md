# Particles

Packs and unpacks `particles.xr`.

```powershell
npm run cli -- particles <command>
```

## Commands

- `particles unpack`: unpack a packed particles file into LTX configs.
- `particles pack`: pack particles configs into a `.xr` file.

## Options

- `-p, --path <path>`: source file or source directory.
- `-d, --dest <dest>`: output file or directory.
- `-v, --verbose`: print verbose logs.
- `-f, --force`: remove an existing output before writing.
